# AyuSethu Web UI
This is the frontend portal for the AyuSethu Supply Chain framework.
Built with Vite, React, Tailwind CSS v4, Framer Motion, and Socket.io.

## Portals Included
- Admin Command Center
- Collector Gateway (Stage 1-5 ML Verification)
- Lab Pharmacognostic Assessment (PDF Generation via IPFS)
- Manufacturer Auction (QR Minting)
- Public Transparency Engine (`/verify/:batchId`)
